# 🚀 Employee Management System

A full-stack Employee Management System built with **React.js**, **FastAPI**, **SQLite**, **SQLAlchemy**, and **Docker**. The application enables efficient employee management through a modern dashboard with analytics and responsive UI.

## ✨ Features

* Employee CRUD Operations
* Search, Filter & Pagination
* Dashboard with KPI Cards
* Donut, Bar, Area & Gauge Charts
* Employee Status & Department Analytics
* Dark/Light Mode
* FastAPI REST APIs
* Docker & Docker Compose Support

## 🛠 Tech Stack

**Frontend:** React.js, React Router, Axios, CSS, Recharts

**Backend:** FastAPI, SQLAlchemy, SQLite, Pydantic

**DevOps:** Docker, Docker Compose, Nginx

## 📂 Project Structure

```
employee-management-system/
│── frontend/
│── backend/
│── docker-compose.yml
│── README.md
```

## 🚀 Run the Project

```bash
docker compose up --build
```

Frontend: `http://localhost:3000`

Backend: `http://localhost:8000`

API Docs: `http://localhost:8000/docs`
